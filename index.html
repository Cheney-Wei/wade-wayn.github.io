hexo+github创建个人博客

前言：本文将使用hexo以及gitHub搭建一个人博客，相比自己申请域名+服务器而言，使用hexo+gitHub主要是免费，便捷。

博文

安装以及相关配置基于windows系统

  1. NodeJS：node.js有诸多用处，此处用来创建hexo博客框架
  2. GIT客户端：将本地的hexo内容提交至远程gitHub仓库

安装

首先安装node.js,建议去node官网下载并进行安装（不在累述），查看node是否安装成功，在DOS中输入命令行：

npm install -g hexo

hexo init

npm install

hexo g

hexo s

Repositories

New

deploy: type: git repo: github: git@github.com:wade-wayn/wadewayn.github.io.git,master

ls -al ~/.ssh

ssh-keygen -t rsa -C “注册GitHub时的邮箱”

ssh-agent -s

eval $(ssh-agent)

ssh-add

clip < ~./.ssh/id_rsa.pub

Ctrl + V

ssh -T git@github.com

yes

hexo g

npm hexo-deployer-git –save

hexo d
```

进行发布

最终在浏览器中输入，之前在GitHub上面的仓库名进行访问，例如：https://wade-wade.github.io/
